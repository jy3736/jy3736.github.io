10 Python 集合（Set）：唯一值與集合運算

課程簡介

本單元介紹 set，它是「無序且元素唯一」的資料結構。你會學會集合的建立、增刪查，以及聯集、交集、差集、對稱差等數學運算，並理解其在成員測試效率上的優勢。

本講義涵蓋主題

1. set 的特性與建立方式

2. 常用操作：新增、刪除、查詢

3. 集合運算：|、&、-、^

4. set 的效率與應用情境

5. 綜合實作：課程選修分析器

1. set 的特性與建立

A. 元素唯一、無序

nums = {1, 2, 2, 3}
print(nums)
names = {"Amy", "Ben", "Amy"}
print(names)
print(type({}))      # dict
print(type(set()))   # set

B. 由其他序列轉成 set

arr = [1, 1, 2, 3, 3]
unique = set(arr)
print(unique)
text = "banana"
chars = set(text)
print(chars)
tuple_data = (10, 20, 20, 30)
print(set(tuple_data))

重點與常見陷阱：

- 空集合要用 set()，不能用 {}。

- set 無序，不保證輸出順序。

2. 基本操作：新增、刪除、查詢

A. 新增與刪除

s = {1, 2}
s.add(3)
print(s)
s = {1, 2, 3}
s.remove(2)
print(s)
s = {1, 2, 3}
removed = s.pop()
print(removed, s)

B. 成員測試與安全刪除

s = {"apple", "banana"}
print("apple" in s)
s = {1, 2, 3}
s.discard(5)  # 不會報錯
print(s)
s = {10, 20, 30}
if 20 in s:
    s.remove(20)
print(s)

重點與常見陷阱：

- remove 找不到元素會錯，discard 不會。

- pop() 移除的是任意元素，不適合依序需求。

3. 集合運算

A. 聯集、交集、差集、對稱差

A = {1, 2, 3}
B = {3, 4, 5}
print(A | B)
A = {1, 2, 3}
B = {3, 4, 5}
print(A & B)
A = {1, 2, 3}
B = {3, 4, 5}
print(A - B)
A = {1, 2, 3}
B = {3, 4, 5}
print(A ^ B)

B. 子集合與相等比較

A = {1, 2}
B = {1, 2, 3}
print(A.issubset(B))
A = {1, 2, 3}
B = {1, 2, 3}
print(A == B)
A = {1, 2, 3}
B = {1, 2}
print(A.issuperset(B))

重點與常見陷阱：

- 運算子語意對應數學集合，閱讀性高。

- 交集空集合代表兩組資料沒有重疊。

4. set 的效率與實務用途

A. 快速 membership testing

allowed = {"admin", "teacher", "student"}
role = "teacher"
print(role in allowed)
blacklist = {101, 202, 303}
uid = 202
if uid in blacklist:
    print("拒絕存取")
seen = set()
for n in [1, 2, 2, 3, 1]:
    if n in seen:
        print("重複：", n)
    seen.add(n)

B. 去重與比較名單

a = ["Amy", "Ben", "Amy", "Cody"]
print(list(set(a)))
class_a = {"A", "B", "C"}
class_b = {"B", "C", "D"}
print("共同學生", class_a & class_b)
required = {"math", "english", "cs"}
selected = {"math", "cs"}
print("尚缺", required - selected)

重點與常見陷阱：

- set 很適合處理大量「是否存在」判斷。

- 若要固定順序輸出，最後可 sorted(set_obj)。

5. 綜合實作：課程選修分析器

A. 實作目標

1. 讀入兩班選課名單。

2. 輸出共同選修、各自獨有名單。

3. 顯示所有選課學生（去重）。

B. 完整程式碼範例

line_a = input("請輸入 A 班名單（空白分隔）：")
line_b = input("請輸入 B 班名單（空白分隔）：")

A = set(line_a.split())
B = set(line_b.split())

common = A & B
only_a = A - B
only_b = B - A
all_students = A | B

print("\n=== 分析結果 ===")
print("共同選修：", sorted(common))
print("A 班獨有：", sorted(only_a))
print("B 班獨有：", sorted(only_b))
print("全部學生：", sorted(all_students))

C. 程式碼亮點解析

1. 直接以集合運算表達需求

common = A & B
only_a = A - B

2. split() + set() 快速去重

A = set(line_a.split())

3. 輸出前排序提升可讀性

print(sorted(common))

D. 擴展練習

1. 增加第三個班級做三集合分析。

2. 顯示對稱差（只選一班的人）。

3. 讀取 CSV 檔案並自動建立集合。
