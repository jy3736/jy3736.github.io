09 Python 二維串列：表格與網格資料處理

課程簡介

本單元介紹二維串列（list of lists），用來表示表格、矩陣、棋盤等資料。你會學會 A[i][j] 存取方式，並透過巢狀迴圈逐列逐欄處理資料。

本講義涵蓋主題

1. 建立與存取二維串列

2. 巢狀迴圈走訪

3. 常見矩陣操作

4. 初始化陷阱與正確寫法

5. 綜合實作：座位表與統計分析

1. 建立與存取二維串列

A. 基本結構

grid = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
print(grid)
matrix = [[10, 20], [30, 40]]
print(matrix[0][1])
board = [[".", ".", "."], [".", "X", "."], [".", ".", "O"]]
print(board[1][1], board[2][2])

B. 更新元素

grid = [[0, 0], [0, 0]]
grid[1][0] = 9
print(grid)
scores = [[80, 90], [70, 60]]
scores[0][1] = 95
print(scores)
seats = [["空", "空"], ["空", "空"]]
seats[0][0] = "已預約"
print(seats)

重點與常見陷阱：

- A[i] 是第 i 列，A[i][j] 才是該列的第 j 欄。

- 先確定列數與欄數再存取，避免索引錯誤。

2. 巢狀迴圈走訪

A. 逐列逐欄列印

grid = [[1, 2, 3], [4, 5, 6]]
for row in grid:
    for cell in row:
        print(cell, end=" ")
    print()
grid = [[1, 2], [3, 4], [5, 6]]
for i in range(len(grid)):
    for j in range(len(grid[i])):
        print(f"grid[{i}][{j}]={grid[i][j]}")
board = [["A", "B"], ["C", "D"]]
for row in board:
    print("|".join(row))

B. 匯總統計

scores = [[80, 90], [70, 60]]
total = 0
for row in scores:
    for s in row:
        total += s
print(total)
scores = [[80, 90], [70, 60]]
row_sums = []
for row in scores:
    row_sums.append(sum(row))
print(row_sums)
grid = [[1, 2, 3], [4, 5, 6]]
max_value = grid[0][0]
for row in grid:
    for v in row:
        if v > max_value:
            max_value = v
print(max_value)

重點與常見陷阱：

- 外層迴圈處理列，內層迴圈處理欄。

- 聚合計算可同時完成，避免重複掃描。

3. 常見矩陣操作

A. 逐元素加總

A = [[1, 2], [3, 4]]
B = [[5, 6], [7, 8]]
C = [[0, 0], [0, 0]]

for i in range(2):
    for j in range(2):
        C[i][j] = A[i][j] + B[i][j]

print(C)
X = [[2, 2, 2], [3, 3, 3]]
Y = [[1, 1, 1], [1, 1, 1]]
Z = []
for i in range(len(X)):
    row = []
    for j in range(len(X[i])):
        row.append(X[i][j] + Y[i][j])
    Z.append(row)
print(Z)
m = [[1, 2], [3, 4]]
for i in range(len(m)):
    for j in range(len(m[i])):
        m[i][j] *= 10
print(m)

B. 轉置（transpose）入門

A = [[1, 2, 3], [4, 5, 6]]
T = [[0, 0], [0, 0], [0, 0]]
for i in range(len(A)):
    for j in range(len(A[0])):
        T[j][i] = A[i][j]
print(T)
A = [[7, 8], [9, 10], [11, 12]]
T = []
for j in range(len(A[0])):
    row = []
    for i in range(len(A)):
        row.append(A[i][j])
    T.append(row)
print(T)
A = [[1, 2], [3, 4]]
T = [[A[j][i] for j in range(len(A))] for i in range(len(A[0]))]
print(T)

重點與常見陷阱：

- 矩陣操作常要求維度一致。

- 轉置後列欄互換，輸出維度會改變。

4. 初始化陷阱與正確寫法

A. 錯誤初始化（共享同一列）

bad = [[0] * 3] * 2
bad[0][0] = 9
print(bad)  # 兩列都被改到
bad = [[False] * 2] * 3
bad[1][1] = True
print(bad)
row = [0, 0]
bad = [row, row]
bad[0][1] = 5
print(bad)

B. 正確初始化

good = [[0] * 3 for _ in range(2)]
good[0][0] = 9
print(good)
grid = []
for _ in range(3):
    grid.append([0, 0])
print(grid)
board = [["."] * 4 for _ in range(4)]
board[2][2] = "X"
print(board)

重點與常見陷阱：

- [[...]*m]*n 對可變內層結構常有共享參考問題。

- 優先用 comprehension 逐列建立。

5. 綜合實作：座位表與統計分析

A. 實作目標

1. 建立 rows x cols 座位表。

2. 依輸入標記已預約位置。

3. 統計剩餘座位並輸出座位圖。

B. 完整程式碼範例

rows = int(input("請輸入列數："))
cols = int(input("請輸入欄數："))

seats = [["O"] * cols for _ in range(rows)]  # O: 空位, X: 已預約

n = int(input("請輸入預約筆數："))
for _ in range(n):
    r = int(input("列（從 0 開始）："))
    c = int(input("欄（從 0 開始）："))
    if 0 <= r < rows and 0 <= c < cols:
        seats[r][c] = "X"

left = 0
for row in seats:
    for seat in row:
        if seat == "O":
            left += 1

print("\n=== 座位圖 ===")
for row in seats:
    print(" ".join(row))
print(f"剩餘空位：{left}")
