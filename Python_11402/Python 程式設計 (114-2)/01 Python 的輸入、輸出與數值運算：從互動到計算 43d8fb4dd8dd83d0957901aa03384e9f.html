01 Python 的輸入、輸出與數值運算：從互動到計算

課程簡介

本單元會建立你對 Python 程式互動流程的第一個完整心智模型：程式如何顯示資訊（print()）、接收使用者輸入（input()）、用變數保存資料，以及使用四則運算（+、-、*、/）算出結果。

完成本課後，你可以寫出小型互動程式：向使用者提問、把文字輸入用 int() 轉成整數，並輸出清楚、可驗證的結果。

本講義涵蓋主題

1. 使用 print() 進行標準輸出

2. 使用 input() 進行標準輸入

3. 變數與數值更新

4. 基本算術運算子（+、-、*、/）

5. 使用 int() 將輸入文字轉為整數

6. 綜合練習：互動式購物總金額計算器

1. 輸出基礎：print()

A. 為什麼輸出很重要

print() 是觀察程式行為最直接的方法。對使用者來說，它能看到結果；對開發者來說，它能協助驗證程式邏輯。

print("歡迎來到 Python")
name = "Alex"
print("使用者：", name)
a = 10
b = 6
print("總和 =", a + b)

重點與常見陷阱：

- 學習階段請多用 print()，確認程式是否照預期執行。

- print("A", "B") 會自動在兩個值中間加一個空白。

B. 讓輸出清楚可讀

可讀性高的輸出，能同時提升除錯效率與使用者體驗。

item = "筆記本"
price = 79
print(f"商品：{item}，價格：${price}")
score = 92
print("考試分數：", score)
print("是否及格：", score >= 60)
x = 7
y = 3
print("x + y =", x + y)
print("x - y =", x - y)

重點與常見陷阱：

- 請加上標籤（例如 "總計 ="），讓讀者一眼看懂數值意義。

- 教學範例避免只印裸值，容易讓初學者看不出脈絡。

2. 輸入基礎：input()

A. 接收使用者資料

input() 會暫停程式，等待使用者輸入。

name = input("請輸入你的名字：")
print("你好，", name)
city = input("請輸入你所在的城市：")
print(f"你住在 {city}。")
first = input("名字：")
last = input("姓氏：")
print(f"全名：{last}{first}")

重點與常見陷阱：

- input() 回傳值永遠是字串（str）。

- 提示文字要具體，使用者才知道該輸入什麼。

B. 理解「輸入先是文字」

在還沒轉型前，輸入值會先以文字方式運作。

a = input("請輸入第一個數字：")
b = input("請輸入第二個數字：")
print(a + b)  # 若輸入 10 與 20，輸出會是 1020
text = input("請輸入任意內容：")
print("資料型別是：", type(text))
age = input("請輸入年齡：")
print("明年年齡（未轉型時會錯）：", age + "1")

重點與常見陷阱：

- 字串相加是串接，不是數值加法。

- 初學除錯時可搭配 type(...) 確認型別。

3. 變數：儲存與重複使用資料

A. 指派值給變數

變數可以把值存起來，方便後續運算與輸出重用。

student = "Mia"
level = 1
print(student, "目前等級是", level)
apples = 5
oranges = 4
total_fruit = apples + oranges
print("水果總數：", total_fruit)
message = "Python"
message = message + " 入門"
print(message)

重點與常見陷阱：

- 變數名稱要有語意（例如 total_price，不要用 tp）。

- = 在程式中代表「指定」，不是數學上的「相等」。

B. 透過運算更新變數

變數可以在運算後被更新為新值。

count = 0
count = count + 1
print(count)
balance = 100
balance = balance - 35
print("剩餘餘額：", balance)
hours = 2
rate = 15
pay = hours * rate
print("薪資：", pay)

重點與常見陷阱：

- 更新步驟建議寫清楚，讓邏輯更容易閱讀與追蹤。

- 同名變數重複使用是合法的，但要留意每次更新後的值。

4. 基本算術運算子

A. 核心運算：+、-、*、/

這些運算子可進行最基本的數值計算。

a = 12
b = 4
print(a + b)  # 16
a = 12
b = 4
print(a - b)  # 8
a = 12
b = 4
print(a * b)  # 48
a = 12
b = 4
print(a / b)  # 3.0

重點與常見陷阱：

- 在 Python 3 中，/ 的結果一定是浮點數（float）。

- 只有在值是數值型別時，算術運算才會符合預期。

B. 在小任務中應用四則運算

price = 120
discount = 25
final_price = price - discount
print("折扣後價格：", final_price)
total = 300
people = 4
print("每人分攤：", total / people)
width = 8
height = 5
area = width * height
print("長方形面積：", area)

重點與常見陷阱：

- 建議使用中介變數（例如 final_price、area）讓公式更清楚。

- 除法結果可能出現小數，即使可整除也會以浮點數表示。

5. 使用 int() 把輸入文字轉成數字

A. 基本整數轉型

要對使用者輸入做數值計算，先把文字轉成整數。

age_text = input("請輸入你的年齡：")
age = int(age_text)
print("明年年齡：", age + 1)
x = int(input("請輸入 x："))
y = int(input("請輸入 y："))
print("x + y =", x + y)
quantity = int(input("請輸入數量："))
unit_price = int(input("請輸入單價："))
print("總金額：", quantity * unit_price)

重點與常見陷阱：

- int("42") 可以成功；但 int("42.5") 會觸發 ValueError。

- 如果輸入是要拿來運算，建議在 input() 後立即轉型。

B. 例外處理：避免不合法整數輸入

try:
    value = int(input("請輸入整數："))
    print("兩倍結果：", value * 2)
except ValueError:
    print("輸入格式錯誤，請輸入整數。")
text = input("請輸入分數：")
if text.isdigit():
    score = int(text)
    print("分數 + 5 =", score + 5)
else:
    print("輸入必須只包含數字字元。")
while True:
    try:
        number = int(input("請輸入正整數："))
        if number > 0:
            print("輸入成功：", number)
            break
        print("請輸入大於 0 的值。")
    except ValueError:
        print("請輸入合法整數。")

重點與常見陷阱：

- 使用者輸入錯誤很常見，程式要主動防護。

- 透過重試迴圈可提升可靠性與使用體驗。

6. 綜合練習：互動式購物總金額計算器

A. 目標

1. 從輸入讀取顧客姓名、商品數量與單價。

2. 驗證數量與單價是否為合法整數。

3. 計算小計、折扣與應付總額。

4. 輸出清楚的結帳摘要。

B. 可直接執行的完整程式

def read_non_negative_int(prompt: str) -> int:
    while True:
        try:
            value = int(input(prompt))
            if value < 0:
                print("請輸入大於等於 0 的整數。")
            else:
                return value
        except ValueError:
            print("輸入格式錯誤，請輸入整數。")


def calculate_discount(subtotal: int) -> int:
    if subtotal >= 1000:
        return 100
    if subtotal >= 500:
        return 50
    return 0


def main() -> None:
    print("=== 購物總金額計算器 ===")

    customer_name = input("顧客姓名：").strip()
    if not customer_name:
        customer_name = "訪客"

    quantity = read_non_negative_int("商品數量：")
    unit_price = read_non_negative_int("商品單價：")

    subtotal = quantity * unit_price
    discount = calculate_discount(subtotal)
    final_total = subtotal - discount

    print("\n--- 收據 ---")
    print(f"顧客：{customer_name}")
    print(f"數量：{quantity}")
    print(f"單價：${unit_price}")
    print(f"小計：${subtotal}")
    print(f"折扣：-${discount}")
    print(f"應付總額：${final_total}")


if __name__ == "__main__":
    main()

C. 程式碼亮點解析

1. 輸入驗證函式

def read_non_negative_int(prompt: str) -> int:
    while True:
        try:
            value = int(input(prompt))
            if value < 0:
                print("請輸入大於等於 0 的整數。")
            else:
                return value
        except ValueError:
            print("輸入格式錯誤，請輸入整數。")

這個函式把「讀取輸入、轉型、驗證、重試」集中管理，主流程會更乾淨。

2. 計算邏輯與輸入輸出分離

def calculate_discount(subtotal: int) -> int:
    if subtotal >= 1000:
        return 100
    if subtotal >= 500:
        return 50
    return 0

折扣規則獨立後，後續測試或調整商業邏輯都更容易。

3. 結構化輸出

print("\n--- 收據 ---")
print(f"顧客：{customer_name}")
print(f"數量：{quantity}")
print(f"單價：${unit_price}")
print(f"小計：${subtotal}")
print(f"折扣：-${discount}")
print(f"應付總額：${final_total}")

格式化輸出能提升可讀性，也方便核對金額。

D. 延伸練習

1. 加入稅額計算（例如 5%），並同時顯示稅前與稅後金額。

2. 支援多個商品項目，最後合併計算總帳單。

3. 新增優惠碼輸入，符合條件時再套用額外折扣。

4. 把每次交易摘要寫入文字檔保存。

結論：你的第一支可靠互動式 Python 程式

- print() 讓結果可視化，也支援除錯。

- input() 用來收資料，但預設是文字。

- 變數可儲存資料並重複使用。

- 算術運算子（+、-、*、/）負責核心計算。

- int() 轉型與輸入驗證是安全處理數值輸入的關鍵。

下一步建議：接著學條件判斷（if / elif / else），讓程式能根據計算結果做決策。
