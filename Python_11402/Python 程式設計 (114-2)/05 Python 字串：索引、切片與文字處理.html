05 Python 字串：索引、切片與文字處理

課程簡介

本單元深入 str 型別。你會學會如何透過索引取得單一字元、用切片擷取子字串、呼叫常見方法（find()、replace()、count()），並理解字串不可變（immutable）的本質。

本講義涵蓋主題

1. 字串索引與負索引

2. 字串切片 [start:stop:step]

3. 常用字串方法

4. 字串格式化與可讀輸出

5. 綜合實作：文字清理與關鍵字分析器

1. 字串索引

A. 正索引與負索引

text = "Python"
print(text[0])
text = "Python"
print(text[-1])
word = "Taiwan"
print(word[2], word[-2])

重點與常見陷阱：

- 正索引從 0 開始。

- 負索引 -1 代表最後一個字元。

B. 安全索引

s = "abc"
print(len(s))
name = "Leo"
if len(name) > 2:
    print(name[2])
msg = ""
if msg:
    print(msg[0])
else:
    print("空字串")

重點與常見陷阱：

- 超出索引範圍會發生 IndexError。

- 索引前可先用 len() 檢查長度。

2. 字串切片

A. 基本切片

text = "PythonProgramming"
print(text[0:6])
text = "PythonProgramming"
print(text[6:])
text = "PythonProgramming"
print(text[:6])

B. 含 step 與反轉

text = "abcdef"
print(text[::2])
text = "abcdef"
print(text[::-1])
text = "0123456789"
print(text[1:9:3])

重點與常見陷阱：

- 切片右邊界 stop 不包含在結果中。

- [::-1] 是常見反轉技巧。

3. 常用字串方法

A. find()、count()、replace()

text = "banana"
print(text.find("na"))
text = "banana"
print(text.count("a"))
text = "I like Java"
print(text.replace("Java", "Python"))

B. 清理與標準化

raw = "  Hello Python  "
print(raw.strip())
name = "ALICE"
print(name.lower())
sentence = "python is fun"
print(sentence.title())

重點與常見陷阱：

- 字串方法通常回傳新字串，不會直接修改原字串。

- 搜尋前先做 strip()、lower() 可提升比對穩定性。

4. 字串不可變與格式化

A. 不可變（immutable）

s = "cat"
# s[0] = "b"  # 這行會錯誤
s = "b" + s[1:]
print(s)
word = "hello"
new_word = word.replace("h", "y")
print(word, new_word)
code = "A001"
code = code + "-TW"
print(code)

B. 格式化輸出

name = "Mia"
score = 93
print(f"{name} 的分數是 {score}")
price = 128.5
print(f"價格：{price:.2f}")
item = "筆電"
qty = 2
print("商品：{}，數量：{}".format(item, qty))

重點與常見陷阱：

- 建議優先使用 f-string，語法簡潔。

- 數值輸出可指定小數位提升報表品質。

5. 綜合實作：文字清理與關鍵字分析器

A. 實作目標

1. 讀入一段使用者文字。

2. 做標準化（去空白、轉小寫）。

3. 統計關鍵字出現次數，並替換敏感詞。

B. 完整程式碼範例

text = input("請輸入一段文字：")
keyword = input("請輸入要查詢的關鍵字：").strip().lower()

clean_text = text.strip().lower()
count = clean_text.count(keyword)
first_pos = clean_text.find(keyword)
filtered_text = clean_text.replace("爛", "*" * 1)

print("\n=== 分析結果 ===")
print(f"清理後文字：{clean_text}")
print(f"關鍵字出現次數：{count}")
print(f"首次出現位置：{first_pos}")
print(f"過濾後文字：{filtered_text}")

C. 程式碼亮點解析

1. 先標準化再分析

clean_text = text.strip().lower()

2. count 與 find 搭配

count = clean_text.count(keyword)
first_pos = clean_text.find(keyword)

3. 以 replace 做簡單文字過濾

filtered_text = clean_text.replace("爛", "*")

D. 擴展練習

1. 支援多個關鍵字統計。

2. 顯示關鍵字前後 5 個字元的上下文。

3. 加入簡單敏感詞清單並批次替換。
