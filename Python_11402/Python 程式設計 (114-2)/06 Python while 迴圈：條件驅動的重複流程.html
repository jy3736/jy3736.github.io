06 Python while 迴圈：條件驅動的重複流程

課程簡介

本單元介紹 while 迴圈。與 for 不同，while 不依固定次數，而是「條件為真就繼續執行」。你會學到 break、continue、sentinel value（哨兵值）與 while ... else。

本講義涵蓋主題

1. while 基本語法

2. break 與 continue

3. 無限迴圈與哨兵值

4. while ... else 的使用情境

5. 綜合實作：密碼重試與鎖定機制

1. while 基本語法

A. 條件為真就持續執行

i = 1
while i <= 5:
    print(i)
    i += 1
countdown = 3
while countdown > 0:
    print(countdown)
    countdown -= 1
print("Go!")
total = 0
n = 1
while n <= 4:
    total += n
    n += 1
print(total)

重點與常見陷阱：

- 必須有「更新條件」的程式碼，避免無限迴圈。

- 先初始化控制變數，再進入迴圈。

B. 驗證輸入直到合法

age = -1
while age < 0:
    age = int(input("請輸入年齡（>=0）："))
print(age)
text = ""
while text == "":
    text = input("請輸入非空字串：").strip()
print(text)
score = -1
while not (0 <= score <= 100):
    score = int(input("請輸入 0~100 分數："))
print(score)

重點與常見陷阱：

- 這類流程適合 while，不適合固定次數的 for。

- 條件寫法要清楚，避免邏輯反轉造成卡住。

2. break 與 continue

A. break：提前離開迴圈

n = 1
while True:
    if n == 4:
        break
    print(n)
    n += 1
while True:
    cmd = input("輸入 q 離開：")
    if cmd == "q":
        break
x = 0
while x < 10:
    x += 1
    if x % 7 == 0:
        print("找到第一個 7 的倍數")
        break

B. continue：跳過本輪剩餘程式

n = 0
while n < 6:
    n += 1
    if n % 2 == 0:
        continue
    print(n)
i = 0
while i < 5:
    i += 1
    if i == 3:
        continue
    print(i)
num = 0
while num < 10:
    num += 1
    if num < 5:
        continue
    print(num)

重點與常見陷阱：

- continue 前後要確認控制變數是否有更新。

- break 過多會讓流程難追蹤，應保留必要出口即可。

3. 無限迴圈與哨兵值

A. 無限迴圈常見框架

while True:
    text = input("輸入內容（exit 離開）：")
    if text == "exit":
        break
    print("你輸入的是：", text)
while True:
    num = int(input("輸入正整數（0 結束）："))
    if num == 0:
        break
    print(num * 2)
while True:
    cmd = input("指令：")
    if cmd in ["quit", "q"]:
        break

B. 哨兵值（sentinel value）

total = 0
while True:
    n = int(input("輸入金額（-1 結束）："))
    if n == -1:
        break
    total += n
print(total)
count = 0
while True:
    line = input("輸入文字（END 結束）：")
    if line == "END":
        break
    count += 1
print(count)
maximum = None
while True:
    n = int(input("輸入數字（999 結束）："))
    if n == 999:
        break
    if maximum is None or n > maximum:
        maximum = n
print(maximum)

重點與常見陷阱：

- 哨兵值應挑選「不會和正常資料衝突」的值。

- 若資料本身可能包含哨兵值，要改用其他結束機制。

4. while ... else

A. 何時會執行 else

while 正常結束（條件變 False）時會執行 else；若用 break 跳出，else 不執行。

n = 1
while n <= 3:
    print(n)
    n += 1
else:
    print("正常結束")
n = 1
while n <= 3:
    if n == 2:
        break
    n += 1
else:
    print("不會印出")
x = 5
while x > 0:
    x -= 1
else:
    print("倒數完成")

重點與常見陷阱：

- while ... else 在搜尋失敗判定時很實用。

- 團隊若不熟這個語法，可用旗標變數替代以提升可讀性。

5. 綜合實作：密碼重試與鎖定機制

A. 實作目標

1. 使用者最多輸入 3 次密碼。

2. 正確就登入成功並停止。

3. 三次錯誤則顯示帳號鎖定。

B. 完整程式碼範例

correct_password = "tw2026"
attempts = 0
max_attempts = 3

while attempts < max_attempts:
    pwd = input("請輸入密碼：")
    if pwd == correct_password:
        print("登入成功")
        break

    attempts += 1
    left = max_attempts - attempts
    print(f"密碼錯誤，剩餘 {left} 次")
else:
    print("嘗試次數過多，帳號已鎖定")

C. 程式碼亮點解析

1. 用條件控制重試上限

while attempts < max_attempts:

2. 成功時即刻 break

if pwd == correct_password:
    break

3. 利用 while ... else 處理失敗收尾

else:
    print("嘗試次數過多，帳號已鎖定")

D. 擴展練習

1. 加入「每次錯誤等待 2 秒」機制。

2. 錯誤次數達 2 次時提示密碼規則。

3. 成功登入後顯示登入時間與歡迎訊息。
