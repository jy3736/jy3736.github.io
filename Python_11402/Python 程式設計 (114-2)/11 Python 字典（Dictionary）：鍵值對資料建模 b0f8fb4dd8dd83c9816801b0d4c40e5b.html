11 Python 字典（Dictionary）：鍵值對資料建模

課程簡介

本單元介紹 dict。字典以 key: value 方式儲存資料，適合表示「名稱對應值」或「ID 對應紀錄」。你會學到新增、更新、刪除、查詢與迭代，並使用 .get()、.items() 等常用方法。

本講義涵蓋主題

1. dict 結構與基本操作

2. 新增、更新、刪除

3. 常用方法：.get()、.keys()、.values()、.items()

4. 迭代與巢狀字典入門

5. 綜合實作：聯絡人管理系統

1. dict 結構與存取

A. 建立字典

student = {"name": "Amy", "score": 88}
print(student)
phone_book = dict(alice="0912", bob="0922")
print(phone_book)
config = {"host": "localhost", "port": 8080, "debug": True}
print(config)

B. 依 key 存取 value

user = {"id": 101, "name": "Leo"}
print(user["name"])
prices = {"apple": 30, "banana": 25}
print(prices["apple"])
profile = {"city": "Taipei", "lang": "zh-TW"}
print(profile["city"])

重點與常見陷阱：

- key 必須可雜湊（hashable），通常用字串、數字、tuple。

- d[key] 若 key 不存在會拋錯。

2. 新增、更新、刪除

A. 新增與更新

scores = {}
scores["Amy"] = 90
print(scores)
scores = {"Amy": 90}
scores["Amy"] = 95
print(scores)
stock = {"pen": 10}
stock.update({"book": 5, "pen": 12})
print(stock)

B. 刪除資料

data = {"a": 1, "b": 2}
del data["a"]
print(data)
data = {"a": 1, "b": 2}
value = data.pop("b")
print(value, data)
data = {"x": 1, "y": 2}
data.clear()
print(data)

重點與常見陷阱：

- 同一個 key 再次賦值會覆蓋舊值。

- 刪除前可先檢查 key 是否存在。

3. 常用方法與安全存取

A. .get() 與預設值

user = {"name": "Amy"}
print(user.get("name"))
user = {"name": "Amy"}
print(user.get("age"))
user = {"name": "Amy"}
print(user.get("age", 0))

B. .keys()、.values()、.items()

d = {"a": 1, "b": 2}
print(list(d.keys()))
d = {"a": 1, "b": 2}
print(list(d.values()))
d = {"a": 1, "b": 2}
for k, v in d.items():
    print(k, v)

重點與常見陷阱：

- .get() 適合避免 KeyError。

- 迭代 items() 可同時拿到 key 與 value。

4. 迭代與巢狀字典

A. 字典迭代模式

scores = {"Amy": 90, "Ben": 75, "Cody": 88}
for name in scores:
    print(name, scores[name])
scores = {"Amy": 90, "Ben": 75, "Cody": 88}
for name, score in scores.items():
    if score >= 80:
        print(name, "優秀")
counts = {}
text = "banana"
for ch in text:
    counts[ch] = counts.get(ch, 0) + 1
print(counts)

B. 巢狀字典

students = {
    "S01": {"name": "Amy", "score": 90},
    "S02": {"name": "Ben", "score": 82}
}
print(students["S01"]["name"])
product = {
    "id": "P100",
    "info": {"name": "Keyboard", "price": 1290}
}
print(product["info"]["price"])
classes = {
    "A": {"count": 30},
    "B": {"count": 28}
}
for cls, info in classes.items():
    print(cls, info["count"])

重點與常見陷阱：

- 巢狀結構適合複合資料，但深度太深會降低可讀性。

- 可配合函式拆解操作，避免主程式過長。

5. 綜合實作：聯絡人管理系統

A. 實作目標

1. 以姓名作為 key，電話作為 value。

2. 支援新增/更新、查詢、刪除、列出。

3. 處理 key 不存在的情況。

B. 完整程式碼範例

contacts = {}

while True:
    cmd = input("指令（set/get/del/list/q）：").strip().lower()

    if cmd == "set":
        name = input("姓名：").strip()
        phone = input("電話：").strip()
        contacts[name] = phone
        print("已新增/更新")

    elif cmd == "get":
        name = input("姓名：").strip()
        phone = contacts.get(name)
        if phone is None:
            print("找不到聯絡人")
        else:
            print(f"{name} -> {phone}")

    elif cmd == "del":
        name = input("姓名：").strip()
        if name in contacts:
            contacts.pop(name)
            print("已刪除")
        else:
            print("找不到聯絡人")

    elif cmd == "list":
        print("\n=== 聯絡人清單 ===")
        for name, phone in contacts.items():
            print(f"{name}: {phone}")

    elif cmd == "q":
        break

    else:
        print("無效指令")

C. 程式碼亮點解析

1. 使用姓名當 key 快速查詢

contacts[name] = phone

2. .get() 安全讀取

phone = contacts.get(name)

3. in 檢查後刪除

if name in contacts:
    contacts.pop(name)

D. 擴展練習

1. 每位聯絡人擴充為巢狀資料（電話、email、地址）。

2. 增加關鍵字搜尋（姓名部分比對）。

3. 將聯絡人資料存入 JSON 檔，啟動時載入。
