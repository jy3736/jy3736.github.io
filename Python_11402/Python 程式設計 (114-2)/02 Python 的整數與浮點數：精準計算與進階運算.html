02 Python 的整數與浮點數：精準計算與進階運算

課程簡介

本單元聚焦 Python 的數值型別：整數（int）與浮點數（float）。你會學會判斷資料型別、正確使用除法相關運算子（/、//、%）、冪次運算（**），以及 math 模組常用函式（如 sqrt()、ceil()）。

完成本課後，你可以寫出更可靠的數值計算程式，並避免常見的型別與精度陷阱。

本講義涵蓋主題

1. int 與 float 的差異

2. 除法、整除、餘數運算

3. 指數運算與運算優先順序

4. math 模組與數學常數

5. 綜合實作：分組與成本估算計算器

1. 數值型別：int 與 float

A. 什麼時候用整數、什麼時候用浮點數

整數適合計數（人數、件數），浮點數適合測量（長度、金額平均值、機率）。

students = 32
print(type(students))
temperature = 26.5
print(type(temperature))
count = 7
average = 7 / 2
print(count, type(count))
print(average, type(average))

重點與常見陷阱：

- 7 是 int，7.0 是 float。

- 有小數點不代表計算一定更精準，反而可能有浮點誤差。

B. 型別轉換

x = 9
print(float(x))
y = 3.99
print(int(y))  # 直接截斷小數，不是四捨五入
raw = "42"
num = int(raw)
print(num + 8)

重點與常見陷阱：

- int(3.99) 會得到 3。

- 轉型前要先確認字串內容是否合法數字。

2. 除法家族：/、//、%

A. 一般除法與整除

print(10 / 3)
print(10 // 3)
price = 550
people = 4
print("每人平均", price / people)
print("每人完整份數", price // people)

重點與常見陷阱：

- / 一律回傳 float。

- // 是向下取整（floor），負數時結果要特別注意。

B. 餘數 % 的應用

print(10 % 3)
minute = 135
hours = minute // 60
left = minute % 60
print(hours, left)
n = 27
print("是否為偶數：", n % 2 == 0)

重點與常見陷阱：

- % 常用於週期問題（時間、輪班、奇偶判斷）。

- 請搭配 // 一起使用，能拆解大部分配額與分組問題。

3. 冪次運算與優先順序

A. 使用 **

print(2 ** 3)
radius = 5
area = 3.14159 * (radius ** 2)
print(area)
print(9 ** 0.5)  # 平方根的另一種寫法

重點與常見陷阱：

- ** 比 *、+ 優先。

- 複雜公式建議加括號提升可讀性。

B. 運算順序與括號

print(2 + 3 * 4)
print((2 + 3) * 4)
score = 80
bonus = 5
weight = 1.2
final_score = (score + bonus) * weight
print(final_score)

重點與常見陷阱：

- 不要靠記憶硬背全部優先順序，遇到混合式就加括號。

- 括號同時是「自我註解」，讓維護更容易。

4. math 模組：函式與常數

A. 常用函式

import math
print(math.sqrt(81))
import math
print(math.ceil(7.01))
print(math.floor(7.99))
import math
angle = math.radians(60)
print(math.sin(angle))

重點與常見陷阱：

- 使用前要先 import math。

- ceil 向上取整、floor 向下取整，請依需求選擇。

B. 數學常數

import math
print(math.pi)
import math
circle_area = math.pi * (3 ** 2)
print(circle_area)
import math
print(math.e)

重點與常見陷阱：

- 常數建議使用 math.pi，不要手寫太粗糙近似值。

- 報表輸出可搭配 round() 或格式化控制小數位。

5. 綜合實作：分組與成本估算計算器

A. 實作目標

1. 接收參加人數與每組上限。

2. 計算需要幾組、最後一組幾人。

3. 輸入總成本後，計算每組平均成本。

B. 完整程式碼範例

import math

people = int(input("請輸入總人數："))
max_per_group = int(input("請輸入每組上限人數："))
total_cost = float(input("請輸入總成本："))

groups = math.ceil(people / max_per_group)
last_group_people = people % max_per_group
if last_group_people == 0:
    last_group_people = max_per_group

avg_cost_per_group = total_cost / groups

print("\n=== 計算結果 ===")
print(f"需要組數：{groups}")
print(f"最後一組人數：{last_group_people}")
print(f"每組平均成本：{avg_cost_per_group:.2f}")

C. 程式碼亮點解析

1. 用 ceil 解決「不整除仍要開新組」

groups = math.ceil(people / max_per_group)

2. 用 % 求最後一組人數

last_group_people = people % max_per_group

3. 輸出格式化控制小數位

print(f"每組平均成本：{avg_cost_per_group:.2f}")

D. 擴展練習

1. 加入「每人最低收費」限制，若低於門檻自動調整。

2. 讓使用者可輸入折扣百分比後再估算成本。

3. 顯示「每組成本整數化後」與「誤差差額」。
