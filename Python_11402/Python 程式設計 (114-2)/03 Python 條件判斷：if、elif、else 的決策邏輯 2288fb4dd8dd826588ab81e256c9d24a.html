03 Python 條件判斷：if、elif、else 的決策邏輯

課程簡介

本單元介紹程式如何「做選擇」。你會使用 if、elif、else 建立分支流程，並結合比較運算子與邏輯運算子（and、or、not）處理真實情境。

完成本課後，你可以設計有條件的互動程式，例如成績分級、登入檢核與資格判定。

本講義涵蓋主題

1. if、elif、else 基本語法

2. 比較運算子與布林值

3. 邏輯運算子：and、or、not

4. 巢狀條件與可讀性

5. 綜合實作：學員資格審核器

1. if、elif、else 基礎

A. 單一條件：if

score = 85
if score >= 60:
    print("及格")
age = 18
if age >= 18:
    print("可投票")
temp = 33
if temp > 30:
    print("今天偏熱")

重點與常見陷阱：

- if 後面必須是布林條件。

- 縮排代表程式區塊，請保持一致（通常 4 個空白）。

B. 多分支：if + elif + else

score = 75
if score >= 90:
    print("A")
elif score >= 80:
    print("B")
elif score >= 70:
    print("C")
else:
    print("D")
hour = 14
if hour < 12:
    print("早安")
elif hour < 18:
    print("午安")
else:
    print("晚安")
password_ok = False
if password_ok:
    print("登入成功")
else:
    print("登入失敗")

重點與常見陷阱：

- 條件順序很重要，先寫最嚴格或最特殊的情況。

- else 是兜底分支，避免漏掉未涵蓋情境。

2. 比較運算子與布林值

A. 比較運算子

x = 10
print(x == 10)
y = 7
print(y != 3)
price = 120
budget = 100
print(price > budget)

重點與常見陷阱：

- == 是比較；= 是指定。

- 比較結果只有 True 或 False。

B. 布林結果驅動流程

member = True
if member:
    print("套用會員價")
stock = 0
if stock <= 0:
    print("缺貨")
name = ""
if name == "":
    print("請輸入姓名")

重點與常見陷阱：

- 可先用 print(condition) 驗證條件真假。

- 請避免過度複雜的一行條件，必要時拆成中介變數。

3. 邏輯運算子：and、or、not

A. and 與 or

score = 82
attendance = 90
if score >= 60 and attendance >= 80:
    print("可通過")
is_weekend = True
is_holiday = False
if is_weekend or is_holiday:
    print("可休息")
age = 16
has_parent = True
if age >= 18 or has_parent:
    print("可報名活動")

重點與常見陷阱：

- and 兩邊都要真才成立。

- or 任一為真就成立。

B. not 與條件反轉

is_raining = False
if not is_raining:
    print("可以出門跑步")
is_banned = True
if not is_banned:
    print("可登入")
else:
    print("帳號停用")
has_error = False
if not has_error:
    print("流程正常")

重點與常見陷阱：

- not 會反轉真假。

- 如果 not 讓語句變難懂，改用正向命名（例如 is_active）。

4. 巢狀條件與可讀性

A. 巢狀條件

age = 20
has_ticket = True
if age >= 18:
    if has_ticket:
        print("可入場")
    else:
        print("請先購票")
else:
    print("未滿年齡")
username = "admin"
password = "1234"
if username == "admin":
    if password == "1234":
        print("登入成功")
    else:
        print("密碼錯誤")
else:
    print("帳號不存在")
score = 91
if score >= 60:
    if score >= 90:
        print("及格且表現優秀")
    else:
        print("及格")
else:
    print("不及格")

重點與常見陷阱：

- 巢狀過深會難維護，可考慮改寫成 elif。

- 建議以需求優先順序安排分支。

5. 綜合實作：學員資格審核器

A. 實作目標

1. 讀取學員分數、出席率與是否完成作業。

2. 依條件判定「通過」、「補件」或「未通過」。

3. 給出明確的審核結果說明。

B. 完整程式碼範例

score = int(input("請輸入測驗分數："))
attendance = int(input("請輸入出席率（0-100）："))
homework_done = input("是否完成作業（yes/no）：").strip().lower() == "yes"

if score >= 60 and attendance >= 80 and homework_done:
    result = "通過"
elif score >= 60 and attendance >= 80 and not homework_done:
    result = "補件後可通過"
else:
    result = "未通過"

print("\n=== 審核結果 ===")
print(f"分數：{score}")
print(f"出席率：{attendance}%")
print(f"作業完成：{homework_done}")
print(f"最終判定：{result}")

C. 程式碼亮點解析

1. 多條件同時判斷

if score >= 60 and attendance >= 80 and homework_done:

2. 用 not 直接寫出反向條件

elif score >= 60 and attendance >= 80 and not homework_done:

3. 先正規化輸入再比較

homework_done = input(...).strip().lower() == "yes"

D. 擴展練習

1. 加入等級制（A/B/C）並依等級顯示建議。

2. 把「最低分數」與「最低出席率」改為可調參數。

3. 增加「重測機制」條件。
