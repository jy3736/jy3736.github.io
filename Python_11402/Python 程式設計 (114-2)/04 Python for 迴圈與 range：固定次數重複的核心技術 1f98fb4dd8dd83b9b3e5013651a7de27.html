04 Python for 迴圈與 range：固定次數重複的核心技術

課程簡介

本單元學習如何讓程式重複執行一段邏輯。你會使用 for 搭配 range() 控制重複次數，理解 start、stop、step 三種參數模式，並用迴圈完成統計與序列產生。

本講義涵蓋主題

1. for 迴圈基本語法

2. range() 一參數、二參數、三參數形式

3. 累加、計數與條件過濾

4. 迴圈巢狀與表格輸出

5. 綜合實作：成績批次分析器

1. for 迴圈基本概念

A. 重複執行固定次數

for i in range(5):
    print(i)
for _ in range(3):
    print("Hello Python")
for i in range(1, 4):
    print(f"第 {i} 次練習")

重點與常見陷阱：

- range(5) 會產生 0,1,2,3,4。

- 若不需要迴圈變數，可用 _ 表示占位。

B. 迴圈變數的角色

for day in range(1, 8):
    print(f"第 {day} 天")
total = 0
for n in range(1, 6):
    total += n
print(total)
for i in range(10):
    if i % 2 == 0:
        print(i)

重點與常見陷阱：

- 迴圈變數每輪都會更新。

- 請避免在迴圈外誤用舊值造成邏輯錯誤。

2. range() 三種常見形式

A. range(stop)

for i in range(4):
    print(i)
for i in range(1):
    print("只跑一次")
count = 0
for _ in range(10):
    count += 1
print(count)

B. range(start, stop)

for i in range(3, 8):
    print(i)
for year in range(2024, 2027):
    print(year)
for score in range(60, 66):
    print(score)

C. range(start, stop, step)

for i in range(0, 10, 2):
    print(i)
for i in range(10, 0, -2):
    print(i)
for n in range(5, 26, 5):
    print(n)

重點與常見陷阱：

- stop 不會被包含。

- step 可為負值，適合倒數。

3. 迴圈中的累加與過濾

A. 累加與平均

total = 0
for n in range(1, 6):
    total += n
print(total)
total = 0
count = 0
for n in range(2, 12, 2):
    total += n
    count += 1
print(total / count)
s = 0
for i in range(100, 106):
    s += i
print(s)

B. 條件過濾

for n in range(1, 11):
    if n % 3 == 0:
        print(n)
for n in range(20):
    if n % 2 == 1:
        print(n)
for n in range(1, 31):
    if n % 5 == 0 and n % 3 == 0:
        print(n)

重點與常見陷阱：

- 累加初始值通常是 0。

- 條件過濾時先確認範圍，再寫條件。

4. 巢狀迴圈

A. 九九乘法表（局部）

for i in range(2, 5):
    for j in range(1, 10):
        print(f"{i} x {j} = {i * j}")
    print("---")
for row in range(3):
    for col in range(4):
        print(f"({row},{col})", end=" ")
    print()
for i in range(1, 4):
    for j in range(1, 4):
        if i == j:
            print("*", end=" ")
        else:
            print("-", end=" ")
    print()

重點與常見陷阱：

- 外層控制大區塊，內層控制細節。

- 巢狀迴圈容易放大運算量，注意效能。

5. 綜合實作：成績批次分析器

A. 實作目標

1. 輸入班級人數。

2. 用 for 重複讀入成績。

3. 輸出總分、平均、最高、最低與及格人數。

B. 完整程式碼範例

n = int(input("請輸入學生人數："))

total = 0
passed = 0
max_score = -1
min_score = 101

for i in range(1, n + 1):
    score = int(input(f"請輸入第 {i} 位學生分數："))
    total += score

    if score >= 60:
        passed += 1
    if score > max_score:
        max_score = score
    if score < min_score:
        min_score = score

average = total / n

print("\n=== 分析結果 ===")
print(f"總分：{total}")
print(f"平均：{average:.2f}")
print(f"最高分：{max_score}")
print(f"最低分：{min_score}")
print(f"及格人數：{passed}")

C. 程式碼亮點解析

1. range(1, n + 1) 對齊人類編號

for i in range(1, n + 1):

2. 單次掃描同時計算多個指標

total += score
if score >= 60:
    passed += 1

3. 以初始邊界追蹤最高與最低

max_score = -1
min_score = 101

D. 擴展練習

1. 額外輸出不及格名單編號。

2. 分數改由一行輸入後拆解處理。

3. 增加區間統計（90+、80-89、60-79、<60）。
