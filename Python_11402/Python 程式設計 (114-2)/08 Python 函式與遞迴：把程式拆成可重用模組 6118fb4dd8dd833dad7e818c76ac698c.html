08 Python 函式與遞迴：把程式拆成可重用模組

課程簡介

本單元學習如何用 def 定義函式，透過參數接收資料、用 return 回傳結果，並理解區域與全域作用域。進階主題會介紹遞迴（函式呼叫自己）與基底條件（base case）。

本講義涵蓋主題

1. 函式定義與呼叫

2. 參數與回傳值

3. 區域與全域作用域

4. 遞迴概念與基底條件

1. 函式定義與呼叫

A. def 基本語法

def greet():
    print("你好，歡迎學習 Python")

greet()
def show_line():
    print("-" * 20)

show_line()
show_line()
def welcome(name):
    print(f"歡迎你，{name}")

welcome("Mia")

重點與常見陷阱：

- 函式要先定義後呼叫。

- 函式名稱應描述行為，如 calculate_total()。

2. 參數與回傳值

A. 參數傳遞

def add(a, b):
    print(a + b)

add(3, 5)
def area_rect(width, height):
    print(width * height)

area_rect(4, 6)
def is_adult(age):
    print(age >= 18)

is_adult(20)

B. return 回傳結果

def add(a, b):
    return a + b

result = add(10, 20)
print(result)
def km_to_mile(km):
    return km * 0.621371

print(km_to_mile(5))
def grade(score):
    if score >= 90:
        return "A"
    if score >= 80:
        return "B"
    return "C"

print(grade(85))

重點與常見陷阱：

- print() 是輸出；return 是把值交還呼叫端。

- 函式若沒寫 return，預設回傳 None。

3. 作用域（Scope）

A. 區域變數（local）

def demo():
    x = 10
    print(x)

demo()
def calc_tax(price):
    rate = 0.05
    return price * rate

print(calc_tax(1000))
def outer():
    msg = "inside"
    print(msg)

outer()

B. 全域變數（global）

counter = 0

def plus_one():
    global counter
    counter += 1

plus_one()
print(counter)
APP_NAME = "ScoreSystem"

def show_name():
    print(APP_NAME)

show_name()
rate = 1.2

def apply(x):
    return x * rate

print(apply(10))

重點與常見陷阱：

- 盡量少用 global，優先用參數與回傳值傳遞資料。

- 常數可放全域，但可變狀態應謹慎管理。

4. 遞迴（Recursion）

A. 基本概念：自己呼叫自己

def countdown(n):
    if n == 0:
        print("完成")
        return
    print(n)
    countdown(n - 1)

countdown(3)
def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n - 1)

print(factorial(5))
def fib(n):
    if n <= 1:
        return n
    return fib(n - 1) + fib(n - 2)

print(fib(6))

B. 基底條件（base case）重要性

def bad(n):
    if n == 0:
        return 0
    return bad(n - 1)

print(bad(3))
def sum_to_n(n):
    if n == 1:
        return 1
    return n + sum_to_n(n - 1)

print(sum_to_n(5))
def reverse_text(s):
    if s == "":
        return ""
    return reverse_text(s[1:]) + s[0]

print(reverse_text("python"))

重點與常見陷阱：

- 遞迴一定要有明確基底條件。

- 深度太深可能觸發遞迴上限，實務常改用迴圈。

5. 綜合實作：階層費率與遞迴計算工具

A. 實作目標

1. 建立函式計算階層費率（分段收費）。

2. 建立遞迴函式計算 1 + 2 + ... + n。

3. 主程式整合並輸出結果。

B. 完整程式碼範例

def calc_fee(hours):
    if hours <= 2:
        return hours * 50
    return 2 * 50 + (hours - 2) * 80


def sum_to_n(n):
    if n == 1:
        return 1
    return n + sum_to_n(n - 1)


h = int(input("請輸入使用時數："))
n = int(input("請輸入 n（n >= 1）："))

fee = calc_fee(h)
series_sum = sum_to_n(n)

print("\n=== 計算結果 ===")
print(f"費用：{fee}")
print(f"1 到 {n} 的總和：{series_sum}")

C. 程式碼亮點解析

1. 函式分工清楚

def calc_fee(hours):

2. 遞迴有明確 base case

if n == 1:
    return 1

3. 主流程只負責 I/O 與整合

fee = calc_fee(h)
series_sum = sum_to_n(n)

D. 擴展練習

2. 額外實作「遞迴版最大公因數」。
