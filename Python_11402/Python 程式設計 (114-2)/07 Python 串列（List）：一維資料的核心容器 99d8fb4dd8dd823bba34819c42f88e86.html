07 Python 串列（List）：一維資料的核心容器

課程簡介

本單元介紹 list，這是 Python 最常用的序列資料結構。你會學會建立串列、存取與更新元素、使用 append() / remove() 等方法，以及如何搭配迴圈處理資料。也會練習 split() 與 join() 在字串與串列之間轉換。

本講義涵蓋主題

1. 建立與存取串列

2. 常用串列方法

3. 迴圈處理串列資料

4. split()、join() 與 list comprehension（入門）

5. 綜合實作：待辦清單管理器

1. 建立與存取串列

A. 建立串列與索引

nums = [10, 20, 30]
print(nums)
names = ["Amy", "Ben", "Cody"]
print(names[0], names[-1])
mixed = [1, "Python", True]
print(mixed[1])

B. 更新與切片

scores = [70, 80, 90]
scores[1] = 85
print(scores)
letters = ["a", "b", "c", "d", "e"]
print(letters[1:4])
data = [1, 2, 3, 4, 5]
print(data[::-1])

重點與常見陷阱：

- 串列可變（mutable），可直接修改元素。

- 索引越界會造成 IndexError。

2. 常用串列方法

A. 新增與刪除

items = ["pen", "book"]
items.append("eraser")
print(items)
nums = [1, 2, 4]
nums.insert(2, 3)
print(nums)
colors = ["red", "blue", "green"]
colors.remove("blue")
print(colors)

B. 查詢與整理

nums = [3, 1, 4, 1, 5]
print(nums.count(1))
nums = [3, 1, 4, 1, 5]
print(nums.index(4))
nums = [3, 1, 4, 1, 5]
nums.sort()
print(nums)

重點與常見陷阱：

- remove(value) 刪第一個符合值。

- sort() 會原地修改；若要保留原資料可用 sorted()。

3. 用迴圈處理串列

A. 逐一走訪

nums = [10, 20, 30]
for n in nums:
    print(n)
names = ["Amy", "Ben", "Cody"]
for i in range(len(names)):
    print(i, names[i])
prices = [99, 120, 35]
total = 0
for p in prices:
    total += p
print(total)

B. 搭配 in 做會員檢查

whitelist = ["alice", "bob", "charlie"]
user = "bob"
print(user in whitelist)
nums = [2, 4, 6, 8]
if 5 not in nums:
    print("5 不在清單中")
fruits = ["apple", "banana"]
key = input("輸入水果名稱：")
if key in fruits:
    print("有庫存")
else:
    print("缺貨")

重點與常見陷阱：

- in 在 list 是線性搜尋，資料量大時要注意效能。

- 累加、計數是最常見的迴圈模式。

4. 字串與串列互轉 + comprehension 入門

A. split() 與 join()

line = "10,20,30"
parts = line.split(",")
print(parts)
words = "Python is fun".split()
print(words)
arr = ["2026", "Spring", "Python"]
result = "-".join(arr)
print(result)

B. list comprehension（入門）

nums = [1, 2, 3, 4]
squares = [x * x for x in nums]
print(squares)
nums = [1, 2, 3, 4, 5, 6]
evens = [x for x in nums if x % 2 == 0]
print(evens)
names = ["amy", "ben", "cody"]
upper_names = [n.upper() for n in names]
print(upper_names)

重點與常見陷阱：

- split() 後元素預設都是字串，必要時要轉型。

- comprehension 適合簡短轉換，過長條件可改回一般迴圈。

5. 綜合實作：待辦清單管理器

A. 實作目標

1. 支援新增、刪除、列出待辦事項。

2. 使用者輸入 q 可結束。

3. 顯示目前項目數與內容。

B. 完整程式碼範例

todos = []

while True:
    cmd = input("指令（add/del/list/q）：").strip().lower()

    if cmd == "add":
        task = input("輸入待辦事項：").strip()
        if task:
            todos.append(task)
            print("已新增")

    elif cmd == "del":
        task = input("輸入要刪除的事項：").strip()
        if task in todos:
            todos.remove(task)
            print("已刪除")
        else:
            print("找不到該事項")

    elif cmd == "list":
        print("\n=== 待辦清單 ===")
        for i, task in enumerate(todos, start=1):
            print(f"{i}. {task}")
        print(f"總數：{len(todos)}")

    elif cmd == "q":
        break

    else:
        print("無效指令")

C. 程式碼亮點解析

1. in + remove 做安全刪除

if task in todos:
    todos.remove(task)

2. enumerate 產生清單序號

for i, task in enumerate(todos, start=1):

3. strip().lower() 正規化使用者指令

cmd = input(...).strip().lower()

D. 擴展練習

1. 增加「編號刪除」模式。

2. 新增「已完成」標記欄位（可改用 dict）。

3. 將清單儲存到檔案，下次啟動可讀回。
