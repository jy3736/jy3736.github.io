Python 程式設計 - 課程概述

本課程以「從實作中建立程式思維」為核心理念，帶領學生由基礎語法出發，逐步建立完整的 Python 程式設計能力。課程強調動手操作與上機實驗，讓學生在撰寫、執行與除錯的過程中理解程式運作邏輯，而非僅停留在語法記憶。

整學期課程將由基礎語法、流程控制與資料結構開始，逐步進入物件導向設計與常用套件應用，最終整合為「迷宮走法模擬」與「機器學習走迷宮」微型專題，讓學生體驗從基礎程式設計到簡易智慧應用的完整流程。

課程強調三個學習目標：

  1. 能獨立撰寫具結構性的 Python 程式
  1. 能運用資料結構與物件導向設計解決問題
  1. 能整合套件與演算法完成實際應用專題

透過循序漸進的上機練習與專題整合，學生將理解：

  • 程式架構如何設計
  • 資料如何被儲存與處理
  • 如何使用函數與模組提升可重用性
  • 如何將演算法轉化為實際可執行的系統

期末採多元評量方式，學生可選擇期末上機考，或完成微型專題展示，強調理論與實作並重。

上課規範

為了營造良好的學習氛圍，本課程將強調「誠信」與「專注」兩大原則。

簽到制度

  • 第一堂課鐘響後 20 分鐘，透過 Zuvio 平台隨機簽到。
  • 缺曠僅影響出席紀錄，不直接影響成績。
  • 嚴禁代簽或遠端簽到，違者扣學期總成績 5 分。

課堂秩序

  • 上課期間須專注於上機操作與課堂任務，不得從事與課程無關活動。
  • 違規處理方式：
    • 第一次：口頭警告
    • 再犯：每次扣總成績 3 分

課程進度表（將依實際教學狀況動態調整）

週次主題核心內容

第3週基本語法及程式架構變數、縮排規則與程式流程

第10週物件導向程式設計 I類別與物件概念
第11週物件導向程式設計 II方法與屬性設計
第12週物件導向程式設計 III類別整合與簡易專案設計

第15週Pandas 基礎應用表格資料處理與分析

第17週運用機器學習走迷宮基礎 ML 概念與應用
第18週期末上機考 / 專題展示成果驗收與展示

課程特色總結

  • 以大量上機操作為主軸
  • 從基礎語法逐步整合至完整系統
  • 結合物件導向與資料分析工具
  • 透過迷宮專題培養演算法思維
  • 強調實作能力與問題解決能力

本課程希望學生在學期結束時，不僅能撰寫 Python 程式，更能設計具邏輯架構的應用系統，並具備將程式應用於資料分析與智慧模擬的能力。
